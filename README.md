# Splunk-soc-lab

## Objective
Simulate a Security Operations Center by monitoring system logs.

## Tools Used
- Splunk

## Steps
1. Installed Splunk
2. Ingested Windows Event Logs
3. Queried login activity

## Findings
- Detected failed login attempts (EventCode 4625)
- Monitored successful logins (EventCode 4624)
- Initially, no failed login events were observed. Test events were generated by intentionally entering incorrect credentials to simulate real-world attack behavior.

## Dashboard
- Created visualization of login attempts

## Key Learning
- Log analysis is critical for detecting threats
- SIEM tools help monitor security events
